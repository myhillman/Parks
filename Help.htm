This application can be used to create multiple WWFF parks from a list in a .csv file.

Line 1 of the CSV file must contain the column header. Below is a list of all columns recognised by the uploader. Other columns, if present, will be ignored.

Subsequent lines must contain data, one line per park, in CSV format. Strings can be enclosed in double quotes if needed.
